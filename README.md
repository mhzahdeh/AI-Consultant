# AI Consultant

AI Consultant is a full-stack consulting workspace demo inspired by the shared Figma Make file.

## Features

- Authentication entry flow with organization-aware sessions
- Dashboard, onboarding, and engagement workspace flows
- Persisted artifacts with version history and save/restore support
- File upload ingestion for txt, md, csv, json, pdf, and docx inputs
- Matched-case workflow for proposal, issue tree, and workplan generation
- Members, invites, billing, usage, and settings flows
- Vault positioning that combines internal knowledge, project artifacts, and public analog cases

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

The Vite app and local API run together from this repository.
